Top Pot Donuts, with franchises all over town, needs your help calculating the number of donuts each location must make every day.
The number of donuts to make depends on the hours of operation—7:00 AM to 6:00 PM—and a few factors unique to each location:

The minimum number of customers per hour.
The maximum number of customers per hour.
The average number of donuts purchased per customer.

Top Pot has asked you to write a program that simulates donut demand.
